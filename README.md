# IPNI.js

Send queries to the International Plant Names Index (https://www.ipni.org/) database.

### Prerequisites	

**axios**: for making GET requests

```	
npm install axios
```	
**mocha**, **chai** & **chai-as-promised**: for running tests
```	
npm install mocha
npm install chai
npm install chai-as-promised
```
